# ウェブサイト制作 ヒアリングシート

クライアント向けのウェブサイト制作ヒアリングフォームです。送信内容は **Google スプレッドシート** に保存できます。

## 使い方

- トップページ（`index.html`）がヒアリングシートです
- **スプレッドシート連携**: [google-apps-script/SETUP_SHEETS.md](google-apps-script/SETUP_SHEETS.md) の手順で、Google Apps Script をデプロイし、`index.html` と `hearing-form.html` の `SPREADSHEET_URL` にウェブアプリの URL を設定してください

## Vercel で公開

このリポジトリを Vercel にインポートすると、そのまま静的サイトとしてデプロイされます。
